# 🚀 Frontend Project - ReactJS (Vite + Tailwind CSS)

Thư mục này chứa mã nguồn giao diện cho bài tập Lập trình Web. Dự án sử dụng **ReactJS**, **Vite** để khởi tạo và **Tailwind CSS v4**.

---

## 🏃 Hướng dẫn khởi chạy nhanh cho Team

Các bạn mở terminal và thực hiện đúng trình tự các lệnh sau:

```bash
# Bước 1: Di chuyển vào đúng thư mục Frontend
cd WEB-PROJECT/frontend

# Bước 2: Cài đặt toàn bộ thư viện (Dependencies)
npm install

# Bước 3: Chạy Server phát triển
npm run dev

🔗 Link truy cập: http://localhost:5173/

## 📂 Cấu trúc thư mục (src)
* **`pages/`**: Chứa các trang chính như `Login.tsx`, `Register.tsx`, `App.tsx`.
* **`components/`**: Chứa các thành phần UI dùng chung (Button, Input, Navbar).
* **`api/`**: Chứa cấu hình Axios để kết nối với Backend Spring Boot.
* **`assets/`**: Chứa hình ảnh và tài nguyên tĩnh của dự án.
